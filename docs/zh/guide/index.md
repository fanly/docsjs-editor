# 指南

欢迎使用 DocsJS Editor 文档。

## 什么是 DocsJS Editor？

DocsJS Editor 是一个多编辑器集成桥接器，可以：

- 使用统一 API 连接多个富文本编辑器
- 运行时在编辑器之间切换，同时保留内容
- 将 DocsJS HTML 快照无缝集成到任何编辑器

## 核心功能

- **运行时编辑器切换**: 在 Tiptap、Quill、CKEditor 等之间运行时切换
- **统一 API**: 单一接口适配所有支持的编辑器
- **DocsJS 集成**: 将 DocsJS HTML 快照导入任何编辑器
- **轻量级**: ~7KB gzipped
- **框架就绪**: React、Vue 和原生 JavaScript 支持
