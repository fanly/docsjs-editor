# 安装

## 环境要求

- Node.js 18+
- 支持的编辑器 (Tiptap、Quill、CKEditor 等)

## 安装

::: code-group

```bash [npm]
npm install @coding01/docsjs-editor
```

```bash [yarn]
yarn add @coding01/docsjs-editor
```

```bash [pnpm]
pnpm add @coding01/docsjs-editor
```

:::

## 依赖

确保已安装 DocsJS：

```bash
npm install @coding01/docsjs
```

## TypeScript

此包使用 TypeScript 编写，包含类型定义。
