# Guide

Welcome to the DocsJS Editor documentation.

## What is DocsJS Editor?

DocsJS Editor is a multi-editor integration bridge that allows you to:
- Connect to multiple rich text editors with a unified API
- Switch between editors at runtime while preserving content
- Seamlessly integrate with DocsJS HTML snapshots

## Key Features

- **Runtime Editor Switching**: Switch between Tiptap, Quill, CKEditor and more at runtime
- **Unified API**: Single interface for all supported editors
- **DocsJS Integration**: Import DocsJS HTML snapshots into any editor
- **Lightweight**: ~7KB gzipped bundle
- **Framework Ready**: React, Vue, and vanilla JS support
