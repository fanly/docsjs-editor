# Installation

## Prerequisites

- Node.js 18+
- A supported editor (Tiptap, Quill, CKEditor, etc.)

## Install

::: code-group

```bash [npm]
npm install @coding01/docsjs-editor
```

```bash [yarn]
yarn add @coding01/docsjs-editor
```

```bash [pnpm]
pnpm add @coding01/docsjs-editor
```

:::

## Peer Dependencies

Make sure you have DocsJS installed:

```bash
npm install @coding01/docsjs
```

## TypeScript

This package is written in TypeScript and includes type definitions.
